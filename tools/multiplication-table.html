How to Use a Multiplication Table

A multiplication table is one of the most powerful learning tools in primary maths. To find the answer to any times-table fact, locate one number along the top and the other down the left side. Where the row and column meet is the product.

This interactive chart lets you tap a number to highlight its entire table, making patterns easy to spot. Try highlighting the 9√ó table and watch how the tens digit goes up while the ones digit goes down!

Patterns to Look For

The table is symmetrical along the diagonal ‚Äî because multiplication is commutative (3 √ó 8 = 8 √ó 3). That means children only need to learn roughly half the facts, since each one has a mirror partner.

The 2√ó column is always even. The 5√ó column ends in 0 or 5. The digits of every 9√ó product add up to 9. Spotting these patterns builds number sense and makes memorisation faster.
